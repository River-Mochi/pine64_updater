# PINE64 Updater

Application for updating PINE64 devices, such as Pinecil.

# Supported devices

- [X] Pinecil V1
- [ ] Pinecil V2
- [ ] PinePhone
- [ ] PinePhone Pro

# Supported platforms

- [X] Windows
- [X] MacOS
- [ ] Linux
- [ ] Android
- [ ] Web

# How to download & install

[Download the newest binaries from the releases page here.](https://github.com/pine64/pine64_updater/releases)
